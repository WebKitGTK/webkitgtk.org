struct WebKitFeatureList {
  /* No available fields */
}

Contains a set of toggle-able web engine features.

The list supports passing around a set of WebKitFeature objects and iterating over them:

g_autoptr(WebKitFeatureList) list = webkit_settings_get_experimental_features();
for (gsize i = 0; i < webkit_feature_list_get_length(list): i++) {
    WebKitFeature *feature = webkit_feature_list_get(list, i);
    // Do something with "feature".
}

Lists of features can be obtained with webkit_settings_get_experimental_features(), webkit_settings_get_development_features(), and webkit_settings_get_all_features().
