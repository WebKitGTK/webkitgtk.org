Several vulnerabilities were discovered in WebKitGTK and WPE WebKit.

* <a name='CVE-2024-4367' href='https://www.cve.org/CVERecord?id=CVE-2024-4367'>CVE-2024-4367</a>
  * Versions affected: WebKitGTK and WPE WebKit before 2.52.5.
  * Credit to Thomas Rinsma of Codean Labs.
  * Impact: A type check was missing when handling fonts in PDF.js, which would allow
    arbitrary JavaScript execution in the PDF.js context. Description: The issue was
    addressed with improved checks.

* <a name='CVE-2026-39872' href='https://www.cve.org/CVERecord?id=CVE-2026-39872'>CVE-2026-39872</a>

  * Credit to Utkarsh Pal, Ignacio Sanmillan (@ulexec).

    crash. Description: The issue was addressed with improved memory handling.
  * WebKit Bugzilla: 313528
