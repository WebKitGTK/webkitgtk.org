This is a bug fix release in the stable 2.52 series.

### What's new in the WebKitGTK 2.52.5 release?

 - Fire scrollend event for instant programmatic scrolls.
 - Increase network idle connection timeout to 115 seconds.
 - Add User-Agent quirk for HBO Max.
 - Fix the build with system malloc.
 - Fix several crashes and rendering issues.

Thanks to all the contributors who made possible this release.
